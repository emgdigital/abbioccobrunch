# Abbiocco Brunch — Demo Site

Full redesign demo for client presentation.

## Pages
- `index.html` — Homepage (hero, about, dishes, reviews, booking form, email capture, loyalty preview, map)
- `carta.html` — Full menu page with sticky category nav
- `fidelitat.html` — Loyalty card programme with FAQ

## Deploy to Vercel

### Option A — Vercel CLI (fastest)
```bash
npm i -g vercel
vercel --prod
```

### Option B — Vercel Dashboard
1. Go to vercel.com → New Project
2. Import from GitHub or drag-and-drop this folder
3. No build step needed (pure HTML/CSS/JS)
4. Deploy → live in ~30 seconds

### Option C — Drag & drop
Go to vercel.com/new and drag the entire `abbioccobrunch` folder.

## Local preview
```bash
npx serve .
# or
python -m http.server 3000
```
